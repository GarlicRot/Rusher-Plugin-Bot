# Contributing

Thank you for your interest in improving Rusher-Plugin-Bot. This project keeps contributions simple and straightforward.

## How to Contribute

1. **Open an Issue**
   - Report bugs
   - Suggest improvements
   - Request new features

2. **Make a Pull Request**
   - Keep changes focused and easy to review
   - Use clear commit messages
   - Update documentation if needed

3. **Follow the Project Structure**
   - Commands are located in `src/commands/`
   - Shared utilities are in `src/utils/`
   - Cached registry files are in `src/cache/`

4. **Test Before Submitting**
   - Ensure the bot starts without errors
   - Verify slash commands load and function correctly

## Coding Style

- Use modern JavaScript (modules, async/await)
- Keep code readable and maintainable
- Reuse functions and utilities rather than duplicating logic

## Changelog

You do not need to update the changelog manually.  
It is automatically generated by GitHub Actions.

## Thanks

Every contribution helps improve the project. Thank you for being part of it.
